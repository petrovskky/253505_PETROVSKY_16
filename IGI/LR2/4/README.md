# MERN_Docker_Application
Docker Compose With Mongo, Express, React, Node (MERN) Application

# Run:
````
    #1: Run project
        docker-compose up
    #2: Check data
        http://localhost:3000/
    #3" Stop Project
        docker-compose down
````
